# Java Practice Programs
This repository contains the Java programs I’ve been practicing while learning different concepts.

I’ve tried to cover basic logic building as well as core topics like OOP, arrays, and strings. As I learn new things, I’ll keep adding more programs here.

# What I’m working on
OOP concepts like inheritance, polymorphism, encapsulation, etc.
Basic programs (odd/even, palindrome, factorial, etc.)
Arrays and string problems.
Exception handling and collections.

# Folder structure
OOP → Programs based on object-oriented concepts.
Basics → Simple logic-based programs.
Arrays → Array-related programs.
Strings → String-related problems.
Advanced → Other concepts like exceptions and collections.

# Why I made this
I created this repository to practice regularly and improve my understanding of Java by writing code, not just reading.

I’ll keep updating it as I continue learning 🙂
